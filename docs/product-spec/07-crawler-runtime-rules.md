# 爬取运行规则

## 活动汪详情页下载

1. 登录与验证码只允许人工完成，系统仅复用本机 `.auth/eventwang.json` 登录态。
2. 进入作品详情页后，先下载当前页面可见详情图。
3. 如需全量文件，必须点击页面官方的“下载文件”按钮进入“确定订单”弹窗。
4. 系统读取弹窗里的“总计/合计”汪币金额：
   - 识别为 `0 汪币`：允许点击“立即下载”，保存官方文件到本地输出目录。
   - 识别为大于 `0 汪币`：停止，不点击“立即下载”。
   - 无法识别金额：停止，不点击“立即下载”。
5. 每次运行必须写入 `manifest.json` 和 `summary.json`，记录图片数量、总页数、剩余页数、订单总计、文件下载状态和本地路径。

## 当前实现入口

- 前端入口：`爬取握手控制台`
- 握手接口：`GET /api/scraping/handshake`
- 活动汪详情接口：`POST /api/scraping/eventwang/detail`
- 执行脚本：`scripts/download-eventwang-design-detail.mjs`

## 禁止动作

- 不自动处理验证码。
- 不在订单总计大于 0 汪币时消耗余额、汪币、券或其他权益。
- 不绕过站内权限、付费墙或下载流程。
- 不对小红书执行验证码绕过、审核规避或自动私信外链引导。
