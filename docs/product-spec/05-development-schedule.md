## 9. 第一版质量门禁

### 内容侧

- 标题不能超过 20 字。
- 正文不能超过 150 字。
- 标签必须 8-12 个。
- 图片必须 6-9 张。
- 素材必须有授权说明。
- 不同账号不能复用发布图片。

### 客服侧

- 不承诺最终报价。
- 不承诺档期。
- 不硬导微信。
- 投诉和负面情绪转人工。
- 合同、发票、付款转人工。

### 工程侧

- TypeScript 无明显类型逃逸。
- 服务端校验所有关键状态流转。
- AI 输出必须 JSON Schema 校验。
- 所有写操作写审计日志。
