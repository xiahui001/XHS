# 页面原型 v1.0

状态：待业务确认。确认前不进入功能代码开发。

## 1. 设计方向

设计名称：运营中控台式工业编辑界面。

目标用户是内容运营、销售跟进人员和老板。界面应该像一个高密度但清晰的运营工作台，而不是营销官网或装饰性 SaaS 首页。

DFII 评估：

| 维度 | 分数 | 说明 |
| --- | --- | --- |
| Aesthetic Impact | 4 | 用窄侧栏、状态色、强表格密度形成记忆点 |
| Context Fit | 5 | 符合矩阵运营、线索跟进、待办处理场景 |
| Implementation Feasibility | 4 | Next.js + Tailwind 可稳定实现 |
| Performance Safety | 4 | 以表格、抽屉、服务端分页为主 |
| Consistency Risk | 2 | 页面多，但组件模式稳定 |

DFII = 15，适合执行。

## 2. 全局布局

采用二段式布局：

- 左侧：固定功能栏，宽 248px。
- 右侧：主内容和操作区域，自适应宽度。
- 右侧顶部：当前页面标题、全局搜索、生成按钮、用户入口。
- 右侧主体：列表、表格、编辑器、抽屉、状态面板。

```text
┌──────────────────────┬──────────────────────────────────────────────────────┐
│ 左侧功能栏            │ 右侧顶部操作栏                                        │
│                      ├──────────────────────────────────────────────────────┤
│ 今日工作台            │ 页面标题 / 筛选 / 主操作按钮                          │
│ 热点参考库            ├──────────────────────────────────────────────────────┤
│ 授权素材库            │                                                      │
│ 草稿库                │ 主内容区：表格、编辑器、详情抽屉、状态面板             │
│ 账号矩阵              │                                                      │
│ 私信助手              │                                                      │
│ 线索列表              │                                                      │
│ Prompt 设置           │                                                      │
│ 审计日志              │                                                      │
└──────────────────────┴──────────────────────────────────────────────────────┘
```

## 4. 页面 1：今日工作台

### 用途

运营每天进入后台后的第一屏，负责启动生成任务、查看今日待处理草稿和线索。

### 右侧内容结构

```text
页面标题：今日工作台
主操作：生成今日 30 篇候选草稿

状态概览：
- 今日候选草稿 30
- 待审核 18
- 已选中 15
- 已发布 0
- 新线索 12

生成入口：
- 关键词输入
- 目标账号多选，默认全选 5 个账号
- 每账号候选数量，默认 6
- 行业场景选择
- 参考库范围选择

今日任务表：
- 账号
- 行业定位
- 候选数
- 已选中
- 素材缺口
- 状态
- 操作
```

### 关键操作

- 输入关键词后创建生成任务。
- 查看生成进度。
- 跳转草稿库筛选当前任务。
- 查看素材缺口。

### 右侧内容结构

```text
顶部筛选：
- 关键词
- 行业账号
- 来源类型
- 创建时间
- 状态

主表格：
- 关键词
- 行业
- 参考标题
- 参考摘要
- 来源链接
- 热度备注

- 创建时间
- 操作

- 参考摘要
```

## 8. 页面 5：草稿详情/编辑

### 用途

运营对单篇草稿做发布前确认。

### 右侧内容结构

```text
左半区：内容编辑
- 标题输入，20 字内计数
- 正文输入，150 字内计数
- 标签编辑，8-12 个计数
- 封面标题候选，3 个
- 咨询引导语

右半区：图片与质量
- 6-9 图排序
- 每张图的用途说明
- 图片替换按钮
- 授权状态提示
- 质量检查结果

底部固定操作：
- 保存
- 重新生成
- 运行检查
- 标记已选中
- 标记已发布
```

### 关键约束

- 字数超限即时提示。
- 标签少于 8 个或多于 12 个不可标记已选中。
- 图片少于 6 张或多于 9 张不可标记已选中。
- 素材授权缺失不可标记已选中。
