## 2. 已锁定设计

- 技术栈：Next.js App Router、TypeScript、Tailwind CSS、shadcn/ui、Supabase。
- 布局：左侧固定功能栏，右侧主内容与操作区域。
- 账号数量：5 个自有小红书账号。
- 账号定位：美业大健康微商、校园、建筑行业、商超、企业年会团建。
- 生成目标：每天 30 篇候选草稿，人工挑选 15 篇发布。
- 草稿规格：6-9 图、20 字内标题、150 字内正文、8-12 标签、3 个封面标题候选。
- 客服范围：私信助手、意图识别、回复建议、官方留资插件线索沉淀。
- 权限范围：第一版后台账号权限相同。
